ENTRADAS

El Daily Standup es una reunión diaria de máximo 15 minutos donde el equipo de desarrollo sincroniza actividades y crea un plan para las próximas 24 horas. Este evento es clave para identificar impedimentos y mantener el flujo de trabajo en el proyecto ZAZDENT.

HERRAMIENTAS

Daily Standup

Propósito

Inspeccionar el progreso hacia el Sprint Goal y adaptar el Sprint Backlog según sea necesario.

Participantes

Todo el equipo Scrum (Desarrolladores, Scrum Master y Product Owner).

Duración

Máximo 15 minutos, preferiblemente a la misma hora y lugar cada día.

Tres preguntas diarias

Pregunta Respuesta
¿Qué hicimos ayer? El día de ayer se finalizó con el procedimiento de login para el cliente; ahora este cliente podrá ingresar de manera correcta para posteriormente realizar su reserva de cita.
¿Qué haremos hoy? Hoy se procederá a refactorizar algunas variables del código y mejorar el diseño. Asimismo, se subirán los nuevos cambios en la rama principal del repositorio.
¿Qué dificultades tuvimos? Se tuvo un poco de dificultad respecto a la integración de las tecnologías para el proceso de login del usuario.
